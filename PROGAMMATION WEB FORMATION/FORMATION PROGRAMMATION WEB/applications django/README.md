# Exercices de formation web

Ce dossier contient deux séries d’exercices :
- Exercices HTML/CSS pour apprendre la base du web
- Exercices HTML/CSS/JS pour découvrir l’interactivité

## Structure
- exercices_html_css : 10 exercices basiques avec HTML et CSS
- exercices_html_css_js : 10 exercices interactifs avec HTML, CSS et JavaScript
